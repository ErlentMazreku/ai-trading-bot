# 🤖 AI Trading Bot

Një trading bot i ndërtuar me Python që përdor strategji bazuar në indikatorët teknikë SMA dhe RSI për të analizuar dhe tregtuar asete financiare.

## 📊 Features
- Të dhëna reale nga Yahoo Finance
- Indikatorët teknikë: SMA 20, SMA 50, RSI
- Strategji automatike blerje/shitje
- Backtest me krahasim vs Buy & Hold
- Dashboard interaktiv me Streamlit

## 🛠️ Tech Stack
- Python 3.11
- pandas, numpy
- yfinance
- scikit-learn
- matplotlib
- Streamlit

## 🚀 Si ta ekzekutosh
1. Instalo libraritë:
pip install pandas numpy matplotlib yfinance scikit-learn streamlit
2. Hap dashboardin:
streamlit run dashboard.py

## 📈 Rezultatet
Boti arriti +30% kthim ndaj Bitcoin buy & hold -21% në periudhën e testimit.
